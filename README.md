# 📊 Sales Data Analysis & Visualization | Python Jupyter Notebook  
**By Yogesh Gadekar**

---

## 📌 Project Overview

This project performs **end-to-end Sales Data Analysis using Python**, focusing on transforming raw data into meaningful insights through data cleaning, exploration, and visualization.

- Analyze customer behavior  
- Identify sales trends  
- Understand product performance  
- Support data-driven decision making  

---

## 🛠️ Tools & Technologies Used

- 🐍 Python (Jupyter Notebook)  
- 📊 Pandas & NumPy – Data Manipulation  
- 📉 Matplotlib & Seaborn – Data Visualization  

---

## 📂 Dataset Description

The dataset contains **11,000+ records** including:

- 👤 User_ID & Customer Name  
- 📦 Product_ID  
- 🎂 Age & Age Group  
- 🚻 Gender  
- 📍 State & Zone  
- 💼 Profession  
- 🛍️ Product Category  
- 📦 Orders & 💰 Sales Amount  

---

## 🧹 Data Cleaning & Preprocessing

- ❌ Removed unnecessary column: *Zipcode*  
- 🔍 Checked dataset shape & structure  
- ⚠️ Handled missing/null values  
- 🔄 Standardized Gender values *(M → Male)*  
- ✔️ Verified data types  

---

## 📊 Exploratory Data Analysis (EDA)

### 🔹 Gender Analysis
- Count plots and pie charts  
- Total sales comparison by gender  

### 🔹 Age Group Analysis
- Distribution of customers across age groups  
- Gender-wise purchasing behavior  

### 🔹 State-wise Analysis
- Top states by orders and revenue  
- Visual comparison using bar charts  

### 🔹 Product Category Analysis
- Most popular product categories  
- Category-wise sales and gender contribution  

### 🔹 Profession Analysis
- Customer professions with highest purchases  
- Top contributing sectors  

### 🔹 Top Products Analysis
- Top 10 products based on order volume  

---

## 📈 Key Insights

- 👩 Female customers contribute more to total sales  
- 🎯 Age group **26–35** dominates purchasing behavior  
- 📍 Top performing states:  
  - Uttar Pradesh  
  - Maharashtra  
  - Karnataka  

- 🛍️ Most popular categories:  
  - Beauty  
  - Sports  
  - Electronics  

- 💼 High-performing professions:  
  - IT  
  - Healthcare  
  - Aviation
